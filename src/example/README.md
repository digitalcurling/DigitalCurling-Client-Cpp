## 思考エンジンクライアント作成例

### rulebased

簡単なif-thenルールに従い、1ターン先のシミュレーションを行う思考エンジンです。

#### ファイル構成

- `rulebased.hpp`
  - 全ルール対応の rulebased クライアントのヘッダーファイル
- `rulebased.cpp`
  - 全ルール対応の rulebased クライアントの実装ファイル

#### アルゴリズム

1. No.1ストーンが敵チームのものならばNo.1ストーンをテイクアウトする。
   No.1ストーンを目標地点とした反時計回り、時計回りのショットについて、それぞれ同数シミュレーションを行い、テイクアウトできた回数が多いほうを採用する。
1. No.1ストーンが自チームのものならば、その2m手前にガードストーンを置く。
1. No.1ストーンがハウス内にないなら、ティーの位置にストーンを置く。
